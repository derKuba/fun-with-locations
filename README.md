# fun-with-locations

small coding kata for fun

this little project contains a frontend and a small backend.

## Install

```
# install fe
npm install

# install be
cd backend
npm install
```

## RUN

```

npm run dev

# backend
cd backend
npm run dev
```

## served

fe runs under: http://localhost:4000
be runs under : http://localhost:3000/search/Neu-Ulm
